Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closes to the equator. To accomplish this analysis, we first pulled data vrom the OpenWeatherAPI to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matlibplot to plot various aspects of the weather vs. latitude. factors we looked at included: temperature, cloudiness, humidity, and windspeed. This site provides the source data and visualizationscreated as part of analysis, as well as explanations and descriptions of any trends and correlations witnessed.
